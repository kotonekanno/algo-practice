## 問題文

カンマ区切りの整数列が1行で与えられる。  
この中から、最小値・最大値・平均値（小数切り捨て）を求めて出力せよ。  
ただし、入力が不正な場合は "error" を出力せよ。

## 入力形式

```
整数,整数,整数,...
```

例：
```
10,20,30
```

## 出力形式

```
最小値,最大値,平均値
```

## 入力例

入力
```
10,20,30
```
出力
```
10,30,20
```

---

入力
```
5
```
出力
```
5,5,5
```

---

入力
```
-10,0,10
```
出力
```
-10,10,0
```

---

入力
```
10,abc,30
```
出力
```
error
```

---

入力
```

```
出力
```
error
```

## ルール

1. 入力は1文字以上
2. カンマで分割してすべて整数であること
3. 空要素禁止
4. 整数は負数OK
5. 平均値は整数（小数切り捨て）

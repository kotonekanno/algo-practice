## 問題文

N 個の品物があります。  
それぞれの品物 i には「重さ wi」と「価値 vi」があります。  
重さの合計が W 以下 になるように品物を選んだとき、得られる価値の最大値を求めてください。  
※ 各品物は 1回までしか選べません（0/1）  

## 入力形式

```
N W
w1 v1
w2 v2
...
wN vN
```

- 1行目：
  - N = 品物の数
  - W = ナップサックの最大重量
- 2行目以降：
  - wi = i番目の重さ
  - vi = i番目の価値

## 出力形式

```
最大価値
```

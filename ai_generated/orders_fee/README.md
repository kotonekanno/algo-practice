## 問題文

あるECサイトで、商品の注文情報が1行の文字列で与えられる。  
この文字列を解析して、合計金額を計算して出力せよ。  
ただし、入力が不正な場合は "error" と出力して終了せよ。  

## 入力形式

```
商品名:単価:数量,商品名:単価:数量,...
```

例：
```
apple:100:2,banana:150:3
```

## 出力

```
合計金額（整数）
```

または

```
error
```

## ルール

1. 各商品は「:」で3つに分割される必要がある
  ```
  商品名
  単価
  数量
  ```
2. 単価・数量は整数であること
3. 単価・数量は0以上
4. 商品は1個以上存在する
5. 合計金額を計算

## 入力例

```
apple:100:2,banana:150:3
```
出力

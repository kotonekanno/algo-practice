## 問題文

システムのアクセスログが1行の文字列で与えられる。  
各ユーザーの利用時間を集計し、最も利用時間が長いユーザーIDとその時間を出力せよ。  
ただし、入力が不正な場合は "error" を出力して終了せよ。  

## 入力形式

```
userId:利用時間,userId:利用時間,...
```

例：
```
A:30,B:20,A:10,C:50
```

## 出力形式

```
ユーザーID:合計時間
```

出力例：
```
C:50
```

## ルール

1. 各要素は「:」で2つに分割できること
2. userIdは英大文字のみ
3. 利用時間は整数（0以上）
4. 入力は1件以上必要
5. 同じuserIdは合算する
6. 最大が複数いる場合、辞書順で小さいuserIdを出力

## 入力例

入力
```
A:30,B:20,A:10,C:50
```
出力
```
C:50
```

---

入力
```
A:30,B:40,C:40
```
出力（BとCが同点 → 辞書順）
```
B:40
```

---

入力
```
A:30,B:-10
```
出力
```
error
```

---

入力
```
A:30,B:abc
```
出力
```
error
```

## 問題文

数字のみからなる文字列が入力される。  
この文字列を連続する同じ数字ごとに圧縮して出力せよ。  
ただし、入力が不正な場合は "error" を出力せよ。  

## 入力形式

数字列（0〜9のみ）

## 出力形式

数字とその連続回数を並べた文字列　　

例：
入力
```
111223
```
出力
```
132213
```

## 入力例

1が3回 → 13  
2が2回 → 22  
3が1回 → 31  

---

入力
```
5
```
出力
```
51
```

---

入力
```
112a3
```
出力
```
error
```

---

入力
```

```
出力
```
error
```

## ルール
1. 入力は1文字以上
2. 数字以外が含まれていたらerror
3. 連続する同じ数字ごとにカウント
